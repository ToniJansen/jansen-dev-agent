System Overview

jansen_dev_agent is an automation architecture made of three intelligent agents that operate without human intervention to optimize software development. An overnight agent performs security reviews and code fixes via GitHub pull requests. A morning agent processes meeting notes to extract action plans and key decisions. A 24/7 Telegram bot handles on-demand file analysis and generates live PDF metrics reports.

The stack runs on Groq's Llama-3.3-70b on a single VPS, prioritizing simplicity and near-zero operational cost. Every AI-suggested change goes through a human triage funnel in GitHub pull requests — ensuring final quality stays under manual control.
